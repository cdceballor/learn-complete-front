Background color:

We have different ways to use the Background:

As a color: using background-color: color

As a image: using background-image: url("address")

repeat and select the direction: using background-repear: repeat-eje

In the same line with all the properties: using background: properties

Border

We have different ways to use the borders:

dotted - Defines a dotted border

dashed - Defines a dashed border

solid - Defines a solid border

double - Defines a double border

groove - Defines a 3D grooved border. The effect depends on the border-color value

ridge - Defines a 3D ridged border. The effect depends on the border-color value

inset - Defines a 3D inset border. The effect depends on the border-color value

outset - Defines a 3D outset border. The effect depends on the border-color value

none - Defines no border

hidden - Defines a hidden border

Padding

We have different ways to use the padding:

top - Defines the top padding, padding-top:20px

right - Defines the right padding padding-right:20px

bottom - Defines the bottom padding padding-bottom:20px

left - Defines the left padding padding-left:20px

all - Defines the top, right, bottom, and left padding padding:20px

horizontal - Defines the left and right padding padding:20px 20px

vertical - Defines the top and bottom padding padding:20px 20px

none - Defines no paddingpadding:0px

Width & height

We have different ways to width and the height: we going to use a general height equal to 100px, only change in the height properties

width - Defines the width of an element width:200px

height - Defines the height of an element height:200px

max-width - Defines the maximum width of an element max-width:200px

max-height - Defines the maximum height of an element max-height:200px

min-width - Defines the minimum width of an element min-width:200px

min-height - Defines the minimum height of an element min-height:200px

Text

letter-spacing - Defines the letter spacing of an element letter-spacing:2

word-spacing - Defines the word spacing of an element word-spacing:2

white-space - Defines the white space of an element white-space:pre-wrap

overflow - Defines the overflow of an element overflow:hidden
