Border

We have different ways to use the borders:

dotted - Defines a dotted border

dashed - Defines a dashed border

solid - Defines a solid border

double - Defines a double border

groove - Defines a 3D grooved border. The effect depends on the border-color value

ridge - Defines a 3D ridged border. The effect depends on the border-color value

inset - Defines a 3D inset border. The effect depends on the border-color value

outset - Defines a 3D outset border. The effect depends on the border-color value

none - Defines no border

hidden - Defines a hidden border

Padding

We have different ways to use the padding:

top - Defines the top padding, padding-top:20px

right - Defines the right padding padding-right:20px

bottom - Defines the bottom padding padding-bottom:20px

left - Defines the left padding padding-left:20px

all - Defines the top, right, bottom, and left padding padding:20px

horizontal - Defines the left and right padding padding:20px 20px

vertical - Defines the top and bottom padding padding:20px 20px

none - Defines no paddingpadding:0px
