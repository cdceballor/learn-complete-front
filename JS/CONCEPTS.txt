--------------------------------------------------------- DOM:----------------------------------------------
Document Objects Models -> Everything inside a page, are the tags that contain the html file, for example
the html as a header, the head and the body as a branches, later, the other tags as p, h1, title, etc.
With DOM you can get the title text, set css, get names of tags, move image
The DOM work as a nodes that we consider are the different tags and we have node relation between them.

Element Node type: 
    The Object has to be unique, normally we use document.getElementById("id") to call something with the id,
    also we have other reserved words like ElementByName, etc
